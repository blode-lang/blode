Blode is a C++-based language built for one purpose: creating efficient, standalone Windows executables with a clean, direct syntax.

View on GitHub Read the Docs
main.bld 
#contain <sysio>

act main() {
    sys.clear();
    sys.output = "Hello from Blode";
    sys.wait(1);
    sys.output = "Done.";
}

Windows-First

Designed to produce native .exe files — no runtimes, no wrappers, no bloat.

Clean Syntax

A minimal syntax that gets out of your way. Familiar enough to pick up fast, distinct enough to be its own thing.

C++ Powered

Built on C++ for performance that matters. Small binaries, fast execution, direct system access.

Built in Public

A solo project in active development. No false promises — just steady, honest work.

Where Blode is today

Blode currently targets simple terminal programs via the sysio library. The core runtime is in place; the language is being shaped around real usage.

Console I/O, screen control, and timed output work. Variables, math, graphics, and a full build pipeline are on the roadmap.
