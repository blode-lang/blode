Blode ist eine minimalistische, ausdrucksstarke Programmiersprache, die entwickelt wurde, um die Barriere zwischen deinen Gedanken und dem Code zu schließen.

Jetzt ausprobieren Beispiel ansehen
hello_world.blode 

// So einfach kann es sein
print("Hallo Welt!")

funktion gruss(name) {
    ergebnis = "Hallo " + name
    rueckgabe ergebnis
}

Einfachheit zuerst

Kein unnötiger Ballast. Blode konzentriert sich auf das Wesentliche, damit du schneller ans Ziel kommst.

Von Menschen gemacht

Nicht für Maschinen optimiert, sondern für die Person, die davor sitzt. Lesbarkeit ist kein Feature, sondern Standard.
