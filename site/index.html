// 01 — language

Syntax that does
exactly what it says.

#contain
Import system modules. No package manager noise — just direct access.
#contain sys
#contain net
#contain fs
act blocks
Every executable unit is an act. Not a function. Not a method. An action.
act fetch(url) {
  let res = net.get(url)
  return res.body
}
sys object
Direct system I/O, timing, and environment — no abstractions in the way.
sys.output("log")
sys.err("fatal")
sys.wait(200)
sys.exit(0)
full program
A complete Blode program — system health check in under 15 lines.
#contain sys
#contain fs

act check(path) {
  if fs.exists(path) {
    sys.output("[OK] " + path)
  } else {
    sys.err("[FAIL] not found: " + path)
  }
}

act main() {
  check("/etc/hosts")
  check("/var/log/sys")
  sys.output("check complete.")
}
// 02 — principles

Built on three
unbreakable laws.

Raw Speed

Blode compiles to lean native bytecode. No GC pauses. No runtime overhead. Your act blocks run at the speed of the machine.

97% native throughput

Action-Logic

Every unit of code is an act. No classes, no decorators, no ceremony. Logic flows through actions — direct, readable, composable.

89% less boilerplate

System-Native

The sys object is a first-class citizen. I/O, timing, signals, and environment are baked in — no third-party wrappers needed.

Zero runtime dependencies
